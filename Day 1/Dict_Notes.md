
-  **Dictionary** =Key - Value pairs (hash maps in some other languages)
```
student = { 'name':'John" , "age":25}
```
- Accessing the value of a key:
```
print(student['name'])
```

    - .get()
    - .update( )
    - .pop()
    - .keys()
    - .items()
